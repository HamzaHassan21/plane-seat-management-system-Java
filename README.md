# Plane Seat Management System (Java)

A console-based Java application developed as part of a university coursework.
The system manages airplane seating, allowing users to buy, cancel, and view seats.

## Features
- Buy and cancel seats
- Display seating plan
- Find first available seat
- Input validation and error handling
- Menu-driven console interface

## Technologies Used
- Java
- Object-Oriented Programming
- 2D Arrays
- Exception Handling

## What I Learned
- Managing state using data structures
- Handling user input safely
- Writing modular and readable Java code
- Applying control flow to real-world problems

## Future Improvements
- Persist data using files or a database
- Convert to a web-based application
- Deploy as a backend service on AWS
